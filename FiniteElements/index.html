The Finite Element Method (FEM)

We use the finite element method to solve partial differential equations describing a boundary value problem. Here, we discuss the solution of the mechanical boundary value problem. The goal is only to briefly introduce the method, and not to discuss the many interesting aspects related to its implementation and use in various settings. This is left for specialized courses.

Weak form

The weak form is an integrated form of the strong form, which makes the problem possible to approximate using the Finite Element Method. To obtain this form, we apply two "tricks":

  1. Scalar multiply the strong form with an arbitrary test displacement field δu‾(x‾)\underline{\boldsymbol{ \delta u}}(\underline{\boldsymbol{ x}})δu​(x​)

  2. Integrate over the domain Ω\OmegaΩ

Because the test displacement field δu‾(x‾)\underline{\boldsymbol{ \delta u}}(\underline{\boldsymbol{ x}})δu​(x​) is arbitrary, the following weak form of the boundary value problem is in practice equivalent to Equation (1). Finite element form

The weak form in Equation (4) is also very difficult to solve, perhaps arguably more difficult than the strong form in Equation (1) since we need to deal with the arbitrary function δu‾\underline{\boldsymbol{ \delta u}}δu​. The main issue for both of these forms, is that the solution is a continuous function u‾\underline{\boldsymbol{ u}}u​. It would be much easier to solve if we could somehow parameterize the function, e.g. if we could write that (in 1D) u(x)=a1x+a2x2+a3x3u(x) = a_1 x + a_2 x^2 + a_3 x^3u(x)=a1​x+a2​x2+a3​x3 and then just determine the coefficients aia_iai​. This is what we will do, namely that we will approximate both u‾\underline{\boldsymbol{ u}}u​ and δu‾\underline{\boldsymbol{ \delta u}}δu​ using so-called base functions:

Base functions

The base functions N‾\underline{\boldsymbol{ N}}N​ have many names, e.g. basis, shape, test, trial, etc. functions. These are generally used interchangably, potentially leading to some confusion. In general, test and trial often denote the functions approximating the test function, δu‾\underline{\boldsymbol{ \delta u}}δu​ (yes, the same name), i.e. N‾δ\underline{\boldsymbol{ N}}^\deltaN​δ. Shape functions often describe the functions approximating the solution u‾\underline{\boldsymbol{ u}}u​, i.e. N‾\underline{\boldsymbol{ N}}N​. Finally, base or basis functions are often used to describe both N‾\underline{\boldsymbol{ N}}N​ and N‾δ\underline{\boldsymbol{ N}}^\deltaN​δ. For the nnn equations fiint=fiextf^{ \mathrm{ int} }_i = f^{ \mathrm{ ext} }_ifiint​=fiext​ above to form a solvable equation system, we require (1) that the base functions are independent. That is, for each j∈[1,n]j\in[1,n]j∈[1,n], there doesn't exist coefficients aia_iai​ where aj=0a_j=0aj​=0 such that N‾j=aiN‾i\underline{\boldsymbol{ N}}_j = a_i \underline{\boldsymbol{ N}}_iN​j​=ai​N​i​. Secondly, we require that the solution is constrained by Dirichlet boundary conditions such that rigid body motions are prevented.

Evaluating the integrals

The force vectors fiintf^{ \mathrm{ int} }_ifiint​ and fiextf^{ \mathrm{ ext} }_ifiext​ in Equation (7) are calculated by solving the integral over the entire domain. But the construction of base functions above ensures that one base function is only non-zero in a few elements, as illustrated below.

This makes it possible to calculate contributions elementwise. If we evaluate N‾i⊗∇‾ \underline{\boldsymbol{ N}}_i\otimes\underline{\boldsymbol{ \nabla}}N​i​⊗∇​, for the base functions above, this becomes a constant tensor as Nj(x‾)N_j(\underline{\boldsymbol{ x}})Nj​(x​) is linear inside the element. Consequently, the strain ϵ‾=[u‾⊗∇‾]sym=ai[N‾i⊗∇‾]sym\underline{\boldsymbol{ \epsilon}} = \left[ \underline{\boldsymbol{ u}}\otimes\underline{\boldsymbol{ \nabla}}\right]^{ \mathrm{sym} } = a_i \left[ \underline{\boldsymbol{ N}}_i\otimes\underline{\boldsymbol{ \nabla}}\right]^{ \mathrm{sym} }ϵ​=[u​⊗∇​]sym=ai​[N​i​⊗∇​]sym is also constant. And since the stress is a function of the strain, then σ(ϵ)\boldsymbol{ \sigma}(\boldsymbol{ \epsilon})σ(ϵ) is constant inside the element, implying that the entire integrand [N‾i⊗∇‾]:σT\left[ \underline{\boldsymbol{ N}}_i\otimes\underline{\boldsymbol{ \nabla}}\right] : \boldsymbol{ \sigma}^{\mathrm{T}}[N​i​⊗∇​]:σT is constant. So for triangular elements with linear base functions, the integrals are trivial to solve. For cases when the base functions are not linear, e.g. quadrilateral elements (4 nodes) or quadratic (6-noded) triangular elements, we need to use numerical integration techniques, see e.g. Niels Ottosen and Hans Petersson, 1992: "Introduction to the Finite Element Method", Chapter 20.
