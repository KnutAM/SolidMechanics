Solid Mechanics

This page contains notes for solid continuum mechanics, covering the following topics

  • Strains

  • Stresses

  • Elasticity

  • The Boundary Value Problem (BVP).

  • Finite Element Method (FEM)

The tensor notation is introduced on a separate webpage.
