Elasticity

The definition of an elastic material model is that the stress, σ\boldsymbol{ \sigma}σ, is a unique function of the strain, ϵ\boldsymbol{ \epsilon}ϵ. That is, if we know ϵ\boldsymbol{ \epsilon}ϵ, we can calculate σ\boldsymbol{ \sigma}σ via the function σ(ϵ)\boldsymbol{ \sigma}(\boldsymbol{ \epsilon})σ(ϵ). Consider the following figures that show different material behaviors

  1. The stress-strain relationship is nonlinear, but for every strain value, there is only one stress value: the material behavior shown is elastic.
  2. The stress-strain relationship is nonlinear, including softening. The same stress value occurs for two different strains, but for each strain value, only a single stress value exists: the material behavior shown is elastic.
  3. After loading the material, the stress-strain curve follows a different path back. The behavior seen here is typical for a damaged material, when the stiffness is reduced due to microscopic cracks in the material. Such behavior is not purely elastic.
  4. Also here the stress-strain curve follows a different path upon unloading. Here, the tangent is initially the same as the first part of the loading curve. Such behavior is typical for plastic material models, and the shown behavior is not purely elastic.

Another class of material behavior not shown above, are rate dependent materials. Typically, polymers have a viscoelastic behavior, and for the same strain we get a higher stress if we increase the strain rate (i.e. the loading rate). If a material is rate dependent, we cannot determine σ\boldsymbol{ \sigma}σ from ϵ\boldsymbol{ \epsilon}ϵ alone (we also need ϵ˙\dot{\boldsymbol{ \epsilon}}ϵ˙), hence this is not an elastic material model.

In general, many materials behave (approximately) elastic within some limits. For example, metals are elastic when the stress is below the yield limit and polymers behave almost elastic if the loading rate is sufficiently slow and the strain is sufficently low.

Hyperelasticity

Hyperelasticity is defined by the existence of a so-called potential Ψ(ϵ)\varPsi(\boldsymbol{ \epsilon})Ψ(ϵ) (a scalar function of the strain), such that

σ=∂Ψ∂ϵ\begin{aligned} \boldsymbol{ \sigma} = \frac{\partial \varPsi}{\partial \boldsymbol{ \epsilon}} \end{aligned}σ=∂ϵ∂Ψ​​

Linear elasticity

The most general form of linear elasticity is given by

σ= E:ϵ\begin{aligned} \boldsymbol{ \sigma} = \textbf{\textsf{ E}}:\boldsymbol{ \epsilon} \end{aligned}σ= E:ϵ​

where E\textbf{\textsf{ E}} E is the 4th order elastic stiffness tensor. This model is hyperelastic provided that E\textbf{\textsf{ E}} E is major symmetric, with Ψ=0.5ϵ: E:ϵ\varPsi = 0.5\boldsymbol{ \epsilon}:\textbf{\textsf{ E}}:\boldsymbol{ \epsilon}Ψ=0.5ϵ: E:ϵ. This is seen by taking the derivative

∂σ∂ϵ= E\begin{aligned} \frac{\partial \boldsymbol{ \sigma}}{\partial \boldsymbol{ \epsilon}} = \textbf{\textsf{ E}} \end{aligned}∂ϵ∂σ​= E​

from Equation (2). Using Equation (1), we get that

∂σ∂ϵ=∂2Ψ∂ϵ∂ϵ= E\begin{aligned} \frac{\partial \boldsymbol{ \sigma}}{\partial \boldsymbol{ \epsilon}} = \frac{\partial^2 \varPsi}{\partial \boldsymbol{ \epsilon} \partial \boldsymbol{ \epsilon}} = \textbf{\textsf{ E}} \end{aligned}∂ϵ∂σ​=∂ϵ∂ϵ∂2Ψ​= E​

Considering this expression in index notation, we have

Eijkl=∂2Ψ∂ϵij∂ϵkl=∂∂ϵij[∂Ψ∂ϵkl]=∂∂ϵkl[∂Ψ∂ϵij]\begin{aligned} \textsf{ E}_{ ijkl} = \frac{\partial^2 \varPsi}{\partial \epsilon_{ij} \partial \epsilon_{kl}} = \frac{\partial}{\partial \epsilon_{ij}}\left[\frac{\partial \varPsi}{\partial \epsilon_{kl}}\right] = \frac{\partial}{\partial \epsilon_{kl}}\left[\frac{\partial \varPsi}{\partial \epsilon_{ij}}\right] \end{aligned} Eijkl​=∂ϵij​∂ϵkl​∂2Ψ​=∂ϵij​∂​[∂ϵkl​∂Ψ​]=∂ϵkl​∂​[∂ϵij​∂Ψ​]​

because we can choose which value to differentiate with first (ϵij\epsilon_{ij}ϵij​ or ϵkl\epsilon_{kl}ϵkl​). Therefore, we have that Eijkl= Eklij\textsf{ E}_{ ijkl}=\textsf{ E}_{ klij} Eijkl​= Eklij​, i.e. that E\textbf{\textsf{ E}} E is major symmetric. Furthermore, since ϵ\boldsymbol{ \epsilon}ϵ is symmetric, E\textbf{\textsf{ E}} E must also be minor symmetric. In 3 dimensions, a 4th order tensor has 34=813^4=8134=81 components. But due to the minor symmetries of E\textbf{\textsf{ E}} E, this is reduced to 62=366^2=3662=36. Furthermore, with the major symmetry there are only 21 remaining independent components, which is the maximum number of possible elastic parameters for a linear elastic material.

Isotropic linear elasticity

The behavior of an isotropic material is independent of the loading direction. So given our function σ=f(ϵ)\boldsymbol{ \sigma} = \boldsymbol{ f}(\boldsymbol{ \epsilon})σ=f(ϵ) for an elastic material, an isotropic elastic material fulfills

Rσ(ϵ)RT=f(RϵRT)\begin{aligned} \boldsymbol{ R}\boldsymbol{ \sigma}(\boldsymbol{ \epsilon})\boldsymbol{ R}^{\mathrm{T}} = \boldsymbol{ f}\left(\boldsymbol{ R}\boldsymbol{ \epsilon}\boldsymbol{ R}^{\mathrm{T}}\right) \end{aligned}Rσ(ϵ)RT=f(RϵRT)​

where R\boldsymbol{ R}R is a proper orthogonal rotation tensor. We use the function f\boldsymbol{ f}f just to highlight the different between the value of σ\boldsymbol{ \sigma}σ and the function σ(ϵ)\boldsymbol{ \sigma}(\boldsymbol{ \epsilon})σ(ϵ). In the case of linear elasticity, this relationship translates into

E=[RT⊗‾RT]: E:[R⊗‾R]\begin{aligned} \textbf{\textsf{ E}} = \left[\boldsymbol{ R}^{\mathrm{T}}\overline{\otimes}\boldsymbol{ R}^{\mathrm{T}}\right] : \textbf{\textsf{ E}} : \left[\boldsymbol{ R}\overline{\otimes}\boldsymbol{ R}\right] \end{aligned} E=[RT⊗​RT]: E:[R⊗​R]​

Linear orthotropic elasticity

While it is possible to have models for anisotropy with 21 independent material paramters as shown above, very few models consider this. Orthotropic material behavior is often the most general form of elastic anisotropy that is assumed. Formally, it is defined by invariance under so-called orthotropic transformations

R^σ(ϵ)R^T=f(R^ϵR^T),R^=I−2n‾(i)⊗n‾(i),n‾(i)⋅n‾(j)=δij\begin{aligned} \boldsymbol{ \hat{R}}\boldsymbol{ \sigma}(\boldsymbol{ \epsilon})\boldsymbol{ \hat{R}}^{\mathrm{T}} = \boldsymbol{ f}\left(\boldsymbol{ \hat{R}}\boldsymbol{ \epsilon}\boldsymbol{ \hat{R}}^{\mathrm{T}}\right), \quad \boldsymbol{ \hat{R}} = \boldsymbol{ I} - 2\underline{\boldsymbol{ n}}^{(i)}\otimes\underline{\boldsymbol{ n}}^{(i)}, \quad \underline{\boldsymbol{ n}}^{(i)}\cdot\underline{\boldsymbol{ n}}^{(j)}=\delta_{ij} \end{aligned}R^σ(ϵ)R^T=f(R^ϵR^T),R^=I−2n​(i)⊗n​(i),n​(i)⋅n​(j)=δij​​

Here, n‾(i)\underline{\boldsymbol{ n}}^{(i)}n​(i) represents the three symmetry planes. Note, that the above relation must only hold for a specific set of symmetry planes for the material to be orthotropic. The transformation tensors fullfill the identity R^=R^−1\boldsymbol{ \hat{R}}=\boldsymbol{ \hat{R}}^{-1}R^=R^−1 and in a coordinate system aligned with n‾(i)\underline{\boldsymbol{ n}}^{(i)}n​(i), the possible R^\boldsymbol{ \hat{R}}R^ are represented as

[−100010001],[1000−10001][10001000−1]\begin{aligned} \begin{bmatrix}-1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1\end{bmatrix}, \quad \begin{bmatrix}1 & 0 & 0 \\ 0 & -1 & 0 \\ 0 & 0 & 1\end{bmatrix} \quad \begin{bmatrix}1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -1\end{bmatrix} \end{aligned}⎣⎡​−100​010​001​⎦⎤​,⎣⎡​100​0−10​001​⎦⎤​⎣⎡​100​010​00−1​⎦⎤​​

But these definitions are hard to grasp the physical implications of. To better understand this, we consider two more physical explanations

  1. There exists three special perpendicular directions. If we apply a normal load in any of these directions, no shear strains will arise.

  2. There exists three special perpendicular directions. Consider a shear strain on a plane perpendicular to one of these directions. If we reverse the shear strain but maintain all other strain components constant, the corresponding shear stress component reverses, while the remaining stress components remain unchanged.

For this specific symmetry, there are 9 independent material parameters. It can be very challenging to determine these if we don't know the orthotropic directions. However, if those directions are known, it is fairly straight-forward to identify the parameters.
